# PRODIGY INFOTECH - DATA SCIENCE INTERNSHIP

## Internship Overview

This repository contains the projects completed during my Data Science Internship at Prodigy InfoTech. The internship focused on developing practical skills in data analysis, data visualization, exploratory data analysis, and machine learning using Python and its data science libraries.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## Tasks Completed

### Task 01: Population Distribution Visualization

**Objective:** Create bar charts and histograms to visualize the distribution of categorical and continuous variables.

**Key Activities:**

* Created population distribution datasets.
* Visualized age and gender distributions using bar charts and histograms.
* Performed basic statistical analysis and graphical representation.

**Skills Gained:**

* Data Visualization
* Exploratory Data Analysis
* Statistical Interpretation

---

### Task 02: Data Cleaning and Exploratory Data Analysis (EDA)

**Objective:** Perform data cleaning and exploratory analysis on the Titanic dataset to identify patterns and relationships.

**Key Activities:**

* Loaded and explored the dataset.
* Checked and handled missing values.
* Generated descriptive statistics.
* Created visualizations for survival analysis, age distribution, and correlations.

**Skills Gained:**

* Data Cleaning
* Exploratory Data Analysis
* Correlation Analysis
* Data Visualization

---

### Task 03: Decision Tree Classifier

**Objective:** Build a machine learning model to predict customer purchase behavior using demographic data.

**Key Activities:**

* Loaded and preprocessed customer data.
* Encoded categorical variables.
* Split data into training and testing sets.
* Built and trained a Decision Tree Classifier.
* Evaluated model performance using accuracy and classification reports.
* Visualized the decision tree.

**Skills Gained:**

* Data Preprocessing
* Machine Learning Classification
* Model Evaluation
* Decision Tree Algorithms

---

### Task 04: Sentiment Analysis

**Objective:** Develop a machine learning model to classify text reviews based on sentiment.

**Key Activities:**

* Preprocessed textual data.
* Converted text into numerical features using CountVectorizer.
* Trained a machine learning model for sentiment classification.
* Evaluated model performance and visualized sentiment distribution.

**Skills Gained:**

* Natural Language Processing (NLP)
* Text Classification
* Feature Extraction
* Machine Learning
