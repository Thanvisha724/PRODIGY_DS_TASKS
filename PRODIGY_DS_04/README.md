# PRODIGY_DS_04 - Sentiment Analysis

Perform sentiment analysis on text reviews using Machine Learning.

## Features
- Text preprocessing with CountVectorizer
- Sentiment classification using Naive Bayes
- Accuracy evaluation
- Sentiment distribution visualization

## Files
- sentiment_data.csv
- task4_sentiment_analysis.py
- requirements.txt

## Run
pip install -r requirements.txt
python task4_sentiment_analysis.py

Output:
- Accuracy score
- Classification report
- Sentiment_Output.png
